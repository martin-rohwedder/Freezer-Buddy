# Freezer Buddy

Tired of going through your entire freezer to see what you have stored?

Freezer buddy is a mobile application, that lets you manage your entire freezer and see what items you have stored and how long it has been stored. Freezer buddy also gives you notifications when an item has been stored for a long time, so you dont forget it.

### Features

- Use AI to help you add items faster to the app
- Get automatic notifications (push and emails), when an item has been stored for a long time, so you dont forget it
- Check your freezer whenever you want, e.g. when you're out shopping, so you know what you have stored
